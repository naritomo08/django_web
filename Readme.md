# django_web

djangoによるWebページ

## 展開方法

```bash
git clone -b devlop https://github.com/naritomo08/django_docker
cd django_docker
cp .env.example .env
git clone git@github.com:naritomo08/django_web.git src
docker-compose build
docker-compose up -d

以下のサイトを参照
http://localhost:8000/
```
